Allow your clients to track their package directly in Wordpress in their order page

== Description ==
With this plugin you can show the package tracking history of an order.

The development of this plugin takes place in
[GitHub](https://github.com/Dazzet/woocommerce-coordinadora). Pull requests and
bug reports are welcome

If you want to add additional features, you can contact marioy47 at gmail.com


== Installation ==
1. Upload \"woocommerce-coordinadora\" folder to \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.

== Frequently Asked Questions ==
= Where do I get and API key and API password for Coordinadora =
You need to be a registered client of _Coordinadora Mercantil_ first.
Then you need to ask for this two pieces of information

= What is the _Client Division_? =
This is also assigned by _Coordinadora Mercantil_ and is selected by them
